# Email Spam Detection using Machine Learning

##  Project Overview

This project is a **Machine Learning-based Email Spam Detection System** that classifies messages as **Spam** or **Not Spam**.

It uses Natural Language Processing (NLP) techniques and a classification algorithm to detect unwanted emails.

##  Features

* Classifies messages into Spam or Not Spam
* Uses **TF-IDF Vectorization** for text processing
* Trained using **Multinomial Naive Bayes**
* Accepts custom user input for real-time prediction
* Displays model accuracy

##  Technologies Used

* Python
* Pandas
* Scikit-learn
* NLP (TF-IDF)

##  Dataset

* Dataset used: **SMS Spam Collection Dataset**
* Contains labeled messages (spam / ham)

##  How It Works

1. Load dataset
2. Clean and preprocess text
3. Convert text into numerical features using TF-IDF
4. Train model using Naive Bayes
5. Predict whether message is spam or not


##  How to Run

1. Install required libraries:

pip install pandas scikit-learn

2. Run the Python file:
   
python spam_detection.py

3. Enter a message when prompted
